# Puzzle 7: The CSS Conundrum

## Puzzle Description

This puzzle is a CSS-themed puzzle. The user is given a snippet of HTML and a CSS selector. They need to figure out what element the selector is targeting and what its `content` property is.

## Prompt for Google AI Studio

"Create a CSS puzzle.
1. Provide a small snippet of HTML with a few nested elements and classes (e.g., `<div class='container'><p class='text'>Here is some text, but a <span class='secret'>hidden</span> word.</p></div>`).
2. Provide a CSS selector that targets a specific element in the HTML (e.g., `div.container p.text span.secret`).
3. Provide a CSS rule for that selector that sets the `content` property to a specific word (e.g., `content: 'password';`).
The puzzle is to find the word in the `content` property."

## Expected Answer/Solution

The word from the `content` property.

## Hint

"The answer is in the style."
