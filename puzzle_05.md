# Puzzle 5: The Encrypted Message

## Puzzle Description

This is a classic substitution cipher puzzle. The user is given an encrypted message and the key to decrypt it.

## Prompt for Google AI Studio

"Create a simple substitution cipher puzzle.
1. Write a short, cryptic sentence (e.g., 'The goat is in the machine').
2. Encrypt the sentence using a simple substitution cipher (e.g., A=B, B=C, etc., or a custom mapping).
3. Provide the encrypted sentence and the key for decryption. The decrypted sentence is the password."

## Expected Answer/Solution

The decrypted sentence.

## Hint

"The alphabet is shifted."
