# Puzzle 8: The Regex Riddle

## Puzzle Description

This puzzle is a riddle that can only be solved by constructing the correct regular expression.

## Prompt for Google AI Studio

"Create a 'Regex Riddle'.
1. Write a short riddle whose answer is a common word (e.g., 'What has an eye, but cannot see?'). The answer is 'needle'.
2. Provide a list of strings, some of which contain the answer to the riddle.
3. The puzzle is to write a regular expression that matches only the strings containing the answer to the riddle. The answer to the riddle is the password to the next level."

## Expected Answer/Solution

The answer to the riddle.

## Hint

"The pattern is the key."
