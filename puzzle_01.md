# Puzzle 1: The Echoing Entrance

## Puzzle Description

This is the first puzzle, designed to be a simple introduction to the labyrinth. The user is presented with a string of text and must reverse it to find the password to the next level.

## Prompt for Google AI Studio

"Generate a short, mysterious message for the entrance of a digital labyrinth. The message should be a single sentence. Then, provide the reversed version of that sentence as the 'password' to the next level. The password should be in all lowercase."

## Expected Answer/Solution

The reversed version of the generated sentence.

## Hint

"The entrance echoes your words back at you."
